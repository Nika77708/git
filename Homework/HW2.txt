1.Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?
git checkout. имя_файла

2.Как вернуть файл из индекса в рабочий каталог?
git restore --staged имя_файла

3.Как отменить ошибочный commit ?
git commit --amend -m "новый commit"

4.Как создать новую ветку и переключится в нее?
git checkout -b название_новой_ветки

5.В чем отличение между fast-forward merge and non fast-forfard merge?
Case 1. В main ветке, по сравнению с веткой, которую мы merge(ым), нет новых коммитов, в таком случае fast-forward merge не создает новый коммит, а скорее объединяет коммит(ы) и последняя версия сливаевом ветки становится main. 
Case 2. В main также есть коммиты, как и ветке, которую мы merge(ым), то no fast-forfard merge объеденит ветки, и оставит при этом main-ветку главной, последний коммит main и последний коммит сливаемой ветки.

6.Можно ли удалить ветку? Как? 
Да, использовав комманду:
git branch -d название_ветки
