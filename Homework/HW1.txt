1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
1.1. Создать репозиторий на GitHub.
1.2. Скопировать SSH-путь данного репозитория.
1.3. Открыть Git Bash, перейти в нужную папку, ввести комманду git clone ssh-путь.  

2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
2.1. Выполнить комманду git init.

3.Как получить на локальный компьютер изменения с github.com
3.1 Выполняя в терминале команду git fetch.

4.Какую информацию нам дают команда :
4.1. git status - отображает состояние рабочего каталога, к примеру, есть ли новые созданные или удаленные папки/файлы .
4.2. git remote -v - команда, которой устанавливается подключение к удаленному серверу и git репозиторию.
